# Getting Test Tokens

Qash provides a testnet faucet that allows you to claim free QASH tokens for testing and exploring the platform. This guide explains how to claim your test tokens and start using Qash on the Miden testnet.

:::warning Important
Qash will not launch a real-world token. The QASH test token is **only for experimentation purposes** on testnet and holds no monetary value. Do not attempt to buy, sell, or exchange the test token or expect any future listings.
:::

![Faucet Icon](/img/faucet/faucet-icon.png)

The Qash faucet gives you **100 QASH test tokens** to get started. These tokens are free and can be used to:

- Test payment features
- Explore payroll functionality
- Experiment with all Qash features on testnet

![Claim Faucet](/img/faucet/claim.png)

:::info Testnet Only
These tokens are only available on the Miden testnet and have no real-world value. They're designed for testing and development purposes only.
:::

Then, click on the `Request free tokens` and wait for a while.

![Claim Faucet](/img/faucet/faucet-waiting.png)

Once it showed success, you should be able to see them in your portfolio as shown below:

![Portfolio](/img/faucet/portfolio.png)
