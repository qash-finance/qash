# Getting Started

1. Clone the repository:
   ```sh
    git clone https://github.com/qash-finance/qash-ui.git
    cd qash-ui
   ```
2. Make sure to install all dependencies by running `pnpm install`
3. Run `cp .env.example .env`
4. Finally, run `pnpm dev:web` to start the dev server
